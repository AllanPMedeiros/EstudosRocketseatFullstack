# Estudos Fullstack Rocketseat

Repositório de estudos do curso Fullstack da Rocketseat.

## Conteúdo

### Fundamentos HTML Básico

- **Primeiros Passos**: Anatomia de tags, aninhamento, caracteres reservados, espaços e quebras de linha, fluxo HTML
- **Elementos Estruturais**: Anatomia de documentos HTML, tags semânticas (header, main, aside, footer, section, article, nav), tags genéricas (div, span)
- **Elementos de Conteúdo**: Títulos e parágrafos, formatação básica, links, listas, imagens, representação de código
- **Atributos**: Atributos HTML, atributos globais (id, class, data, style), atributos booleanos
- **Práticas**: Projetos práticos aplicando os conceitos aprendidos

## Objetivo

Documentar o aprendizado e evolução no desenvolvimento web fullstack, com foco inicial em HTML e suas funcionalidades.

Devido à experiência na faculdade de ADS não ter sido satisfatória, se faz necessário reaprender a tríade do front-end (HTML, CSS e JavaScript) desde o início, construindo uma base sólida para o desenvolvimento web.

Este repositório será atualizado constantemente conforme for aprendendo novas tecnologias como CSS, JavaScript, Node.js, React e outros frameworks.
